# 🦊 FOX DESIGN | MAESTRO WEB EDITION V6.0

O Maestro Web Edition é o coração tecnológico da **Operação Raposa**. Uma aplicação de edição de vídeo automatizada, orquestrada por IA, focada na criação de Reels e Shorts de alta fidelidade para a Fox Design.

## 🚀 Arquitetura de Elite (Esquadras Fox)

- **Lúmen Fox:** Especialista visual responsável pelo redimensionamento inteligente (9:16) e aplicação de branding (Logo).
- **Scribe Fox:** Especialista em linguagem. Utiliza IA Whisper para transcrição sincronizada e gera legendas Ultra-HD com estética CSS Pro.
- **Fluxo Fox:** Maestro do áudio. Realiza o "Smart Jump-Cut" (remoção automática de silêncios) e mixagem de trilha sonora.
- **Guardião Fox:** Sistema de segurança e gestão de arquivos temporários.

## ✨ Diferenciais Tecnológicos

- **Sync Engine:** O sistema realiza o Jump-Cut antes da transcrição, garantindo sincronia milimétrica entre fala e legenda.
- **Zero-Crop Engine:** Legendas com Auto-Scale e Padding Vertical, garantindo que nenhuma letra seja cortada na base ou nas laterais.
- **Ultra-HD Render:** Exportação calibrada em 25M bitrate com presets de nitidez máxima para redes sociais.
- **Web Interface:** Controle total via interface Glassmorphism moderna com preview em tempo real.

## 🛠️ Tecnologias Utilizadas

- **Backend:** Python, Flask, MoviePy v2.2.1, OpenAI Whisper.
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (SSE para logs em tempo real).
- **Processamento:** FFmpeg (libx264).
